# Hero background slideshow images

Put homepage background photos in this folder.

Use these file names:

- `coffee-1.jpg`
- `coffee-2.jpg`
- `coffee-3.jpg`
- `coffee-4.jpg`
- `coffee-5.jpg`

PNG and WebP also work:

- `coffee-1.png`
- `coffee-1.webp`

The homepage will automatically find existing images with these names and switch them every 4.5 seconds.

Recommended image size: 1600px wide or larger.
