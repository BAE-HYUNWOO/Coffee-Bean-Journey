# Chapter 4 数据描述：全球咖啡消费

## 数据来源

本模块所使用的咖啡消费数据来源于 Kaggle 公开数据集：  
**[ICO Coffee Dataset Worldwide](https://www.kaggle.com/datasets/yamaerenay/ico-coffee-dataset-worldwide)**

该数据集由用户 `yamaerenay` 整理，原始数据源自 **国际咖啡组织（International Coffee Organization, ICO）** 官方统计。ICO 是联合国下属的政府间组织，其发布的咖啡生产、消费、贸易、库存等数据是全球咖啡行业的权威参考。

## 数据集概览

- **更新时间**：约5年前（历史数据，不包含最新年份）
- **文件格式**：CSV
- **数据范围**：涵盖多个国家/地区自1990年以来的年度咖啡数据
- **数据完整性**：经过清洗，结构规整，适合数据可视化与分析

## Chapter 4 使用的文件

在完整数据集中，与“咖啡消费”相关的数据主要来自以下两个CSV文件：

| 文件名 | 说明 | 本章使用情况 |
| :--- | :--- | :--- |
| `disappearance.csv` | “咖啡消失量”，通常被视为国内总消费量的近似指标（包括损耗） | 主要使用 |
| `domestic-consumption.csv` | 明确的国内消费量数据 | 作为补充或交叉验证 |

> 注：不同国家的统计口径可能略有差异。在可视化中我们优先采用 `disappearance` 字段，并注明其含义为“总国内消费（含损耗）”。

## 关键字段说明

以下是从上述文件中提取的、用于Chapter 4可视化的核心字段：

| 字段名 | 描述 | 示例值 |
| :--- | :--- | :--- |
| `country` | 国家/地区名称 | United States |
| `country_code` | ISO 3字母国家代码 | USA |
| `year` | 咖啡年份（通常为公历年份） | 2018 |
| `disappearance` | 国内消失量（千袋，每袋60公斤），代表总消费量 | 26150 |
| `domestic_consumption` | 国内消费量（千袋），较严格的消费口径 | 25800 |
| `population` | 人口数（千人） | 327000 |
| `total_imports` | 总进口量（千袋） | 24700 |

此外，为计算人均消费（公斤/人/年），可使用以下公式：
人均年消费量（公斤）= (消失量（千袋） × 60 公斤/袋) / 人口（千人）

## 数据时效性声明

⚠️ **重要提示**  
本数据集为**历史数据**，最后更新距今约5年，不包含最近的咖啡年份。  
因此，本项目展示的消费趋势、排行榜等图表反映的是**历史格局**，并非当前实时数据。

在项目页面中，我们将通过以下方式保持透明：

- 图表标题或副标题中标注：`数据来源：ICO（截至20XX年）`
- 在结论部分注明数据为历史趋势分析，不包含最新变化
- 如需引用最新行业动态，将单独标注其他来源（如USDA、WorldAtlas等）

## 数据预处理说明

为了简化前端开发，实际使用的 `consumption.csv` 文件是从原始数据中派生生成的，主要做了以下处理：

- 筛选出 `year >= 2000` 的数据（聚焦近期趋势，避免过早期数据缺失过多）
- 保留每年消费量排名前30的国家/地区
- 计算人均消费量（公斤/年），并四舍五入保留一位小数
- 统一国家名称与ISO代码（参考World Bank标准）
- 缺失值以空字符串表示，由前端图表库忽略

派生后的 `consumption.csv` 字段示例：

| country | iso3 | year | total_consumption_k_bags | per_capita_kg |
| :--- | :--- | :--- | :--- | :--- |
| United States | USA | 2018 | 26150 | 4.8 |
| Brazil | BRA | 2018 | 22500 | 6.2 |
| Germany | DEU | 2018 | 8600 | 7.1 |
